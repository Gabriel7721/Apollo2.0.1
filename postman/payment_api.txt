### create Address
POST http://localhost:9999/api/payments/address
Content-Type: application/json

{
  "id": 0,
  "userId": 0,
  "district": "",
  "ward": "",
  "city": "",
  "street": "",
  "defaultAddress": false
}
------------------------------------------------------
Status 201
{
    "id": 1,
    "district": "Binh Chanh",
    "ward": "Binh Hung",
    "city": "Ho Chi Minh",
    "street": "Nguyen Van Linh"
}

### Create Payment Method (visa/master cards...)
POST http://localhost:9999/api/payments/payment-method
Content-Type: application/json

{
  "id": 0,
  "userId": 0,
  "cartNumber": 0,
  "nameOnCard": "",
  "expirationDate": "",
  "defaultPayment": false
}
--------------------------------------------------------
Status 201
{
    "id": 1,
    "cartNumber": 113999311,
    "nameOnCard": "Gabriel Belmont",
    "expirationDate": "23/06/2034",
    "defaultPayment": true
}
-------------------------------------------------------------------------------------------------------------------------------------
-------------------------------------------------------------------------------------------------------------------------------------
